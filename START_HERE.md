## 🎯 এখনই করার কাজ (15 মিনিট)

### ১️⃣ Billing Account যুক্ত করুন (5 মিনিট)

**কেন?** Firestore database তৈরি করতে billing account লাগে।

**কিভাবে?**
```
Firebase Console → Project Settings → Billing → Link billing account
```

**বিস্তারিত গাইড:** `BILLING_SETUP_STEPS.md` পড়ুন

---

### ২️⃣ Firestore Database তৈরি করুন (5 মিনিট)

**কিভাবে?**
```
Firebase Console → Firestore Database → Create database
→ Start in test mode → Select region → Create
```

**অপেক্ষা করুন:** ২-৩ মিনিট

---

### ३️⃣ Products Collection তৈরি করুন (2 মিনিট)

**কিভাবে?**
```
Firestore → Start collection → ID: "products" → Next → Auto ID → Save
```

---

### ४️⃣ প্রথম Product যোগ করুন (2 মিনিট)

**দুটি উপায়:**

**উপায় ১: Admin Panel (সহজ)**
```
http://localhost:5173/admin
```

**উপায় ২: Firebase Console**
```
Firestore → products collection → Add document
```

---

```
http://localhost:5173
```
